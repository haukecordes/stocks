Anleitung zum Vergleichen von Aktien:

Anforderungen:
- Python 3.8 oder neuer muss installiert sein. Der Befehl "python --verion" sollte die Python Version anzeigen.
- 2 Python Module: "requests, pandas" 
	--> per Befehl in der Command-Line installieren:

		"python -m pip install -r requirements.txt"

Öffne die Command-Line, indem du cmd in die Windows-Suchleiste eingibst und Enter drückst.
Eventuell musst du auch dein Python Verzeichnis finden und hier die Command-Line öffnen.
Um das Programm zu starten führe einen Doppelklick auf die "compare-stocks.py" - Datei aus oder 
gib "python compare-stocks.py" in die Command-Line ein.

Du hast es geschafft! :)
